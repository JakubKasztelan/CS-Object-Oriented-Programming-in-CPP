# CS: Object-Oriented Programming in C++
This repository serves as an archive for projects and assignments completed during the Object-Oriented Programming (OOP) course. The contents demonstrate the application of C++ principles such as encapsulation, inheritance, polymorphism, and template programming.

## Prerequisites
To build and run the projects in this repository, ensure your development environment includes the following:
- **C++ Compiler:** A compiler supporting at least C++17 (e.g., GCC 7+, Clang 5+, or MSVC 2017+).
- **CMake:** Version 3.10 or higher is required to manage the build process and generate build files.
- **Build Tool:** A native build system such as GNU Make (Linux/macOS) or MSVC (Windows) to execute the commands generated by CMake.

### Compilation and Execution

This project uses **CMake** for build automation. To compile and run the project, follow these steps:

1. **Create a build directory:**
```Bash
mkdir build && cd build
```
2. **Configure the project:**
```Bash
cmake ..
```

**3. Build the executable:**
```Bash
cmake --build .
```

**4. Run the demo:**
```Bash
./filename
```
